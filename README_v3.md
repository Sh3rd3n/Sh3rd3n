I'm a Senior Full Stack Developer based in Turin, Italy, with Sardinian roots and 6+ years of shipping production code. I like building things ‚Äî always have. SvelteKit is my framework of choice, Bun is my runtime, Tailwind makes things pretty, and Zod makes sure `any` is not a type ‚Äî because `any` is not a type, it's a cry for help. I also do the enterprise stuff when I have to ‚Äî React, Next.js, Java Spring Boot, Angular, the whole circus. The Venn diagram of "things I enjoy" and "things that pay well" is not a circle, but there's overlap.

I build fast and ship clean ‚Äî velocity without cutting corners. I care more about whether something works in production than whether the architecture diagram looks clean in Confluence. If you're a startup that moves fast, we'll get along. If you need someone to sit in a 3-hour sprint planning meeting to estimate story points, I'm probably not your guy.
